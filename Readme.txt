1) Запоминает предпочтения пользователя (ценовая группа). Поменять можно, только если выбрать из списка комплектов предложение из другой ценовой группы.

2) Меняя тип освещения или длину ленты, делает первым предложение из предпочтений.

3) Меняя цвет, ищет цвет в выбранной ленте, не найдя, переходит к другой из списка.

4) Если вдруг пользователь получит старую ссылку с не актуальными параметрами, система адаптируется.
